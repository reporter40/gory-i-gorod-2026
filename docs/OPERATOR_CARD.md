## Если что-то не так на экране

**Шаг 1** — не паникуй. Числа застыли — зрители всё равно видят данные.

**Шаг 2** — попробуй по порядку:
```
1. F5 на странице /pulse
2. Открыть /pulse в новой вкладке (инкогнито)
3. F5 на /pulse/admin
```

**Шаг 3** — если мигает или глючит → **FREEZE**:
1. В `/pulse/admin` нажать **"🚨 EMERGENCY FREEZE"**
2. Подтвердить → экран застывает, выглядит рабочим
3. Аудитория ничего не замечает
4. Когда починишь → **UNFREEZE**

**Шаг 4** — если `/pulse` совсем не загружается:
1. Открыть файл `pulse-snapshot-XXXXXX.html` в браузере (файл на флешке / в Telegram)
2. Переключить проектор на это окно
3. Звонить разработчику
