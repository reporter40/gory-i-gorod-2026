**Источники:** `scripts/xlsx_full_dump.txt`, строки 40–71 листа «Основная программа»; продакшен https://gory-i-gorod-2026.vercel.app/program; код `lib/data.ts` (`SESSIONS` с `day === 2`).

## Как устроена страница `/program`

Страница — клиентский компонент (`'use client'`, `useState(1)`). При запросе `curl`/без JS в разметке рендерится **только активная вкладка по умолчанию — день 1** (карточки 16 мая). Контент дня 2 появляется **после выбора «День 2 · 17 мая»** в браузере; источник данных тот же — `getSessionsWithSpeakers()` из `lib/data.ts`.

**Параллельные треки (день 2):** слоты «КубГУ» и «Главный зал» в одни и те же часы в XLSX — разные площадки. В UI список после фильтра по дню сортируется **по `starts_at`, затем по `hall` (locale `ru`)** — см. `app/program/page.tsx`; при одинаковом времени порядок залов алфавитный (например, «Главный зал» перед «КубГУ»).
